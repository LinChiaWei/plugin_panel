# Grafana panel plugin

## Getting started

### Start up

1. Install dependencies

   ```bash
   npm install @emotion/css
   npm install
   ```

2. Build plugin in production mode

   ```bash
   npm run build
   ```


3. Build docker image

   ```bash
   docker build
   ```

4. Start the services
    ```bash
   docker-compose up -d
   ```
